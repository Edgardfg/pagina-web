Tipos de paginas web!

Página Estática

Son páginas web básicas que muestran información fija y no cambian según la interacción del usuario.

Página Dinámica

Son páginas web que pueden cambiar el contenido en función de las acciones del usuario, la información de la base de datos u otros factores.

Perfiles de Programadores

1. Programador Frontend

El programador frontend se encarga del desarrollo de la interfaz de usuario de una aplicación web. Utiliza principalmente HTML, CSS y JavaScript.

2. Programador Backend

El programador backend se concentra en el desarrollo de la lógica de la aplicación que se ejecuta en el servidor. Suele utilizar lenguajes como Python, Java, PHP o Node.js.

3. Full Stack Developer

El full stack developer es capaz de trabajar tanto en el frontend como en el backend de una aplicación. Tiene conocimientos en una amplia variedad de tecnologías.

HTML

    "HyperText Markup Language" (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear páginas web. Es un lenguaje de marcado que define la estructura y el contenido de una página web mediante etiquetas y elementos. Estas etiquetas permiten al navegador web interpretar y mostrar el contenido de manera adecuada al usuario. En HTML, las etiquetas se utilizan para marcar diferentes partes del contenido, como encabezados, párrafos, imágenes, enlaces, formularios, listas y más. Cada etiqueta tiene una función específica y puede tener atributos que proporcionan información adicional sobre el elemento.

Estructura del Código HTML

    DOCTYPE hTML:Es la etiqueta que define que el documento es HTML.

    html: Este elemento envuelve todo el contenido de la página y define que el documento es un documento HTML.

    head: Esta sección contiene metadatos sobre el documento, No se muestra en la ventana del navegador.

    meta charset="UTF-8": Esta etiqueta especifica la codificación de caracteres utilizada en el documento. UTF-8 es la codificación recomendada y es compatible con una amplia gama de caracteres de diferentes idiomas.

    title: Define el título de la página que se muestra en la pestaña del navegador o en la barra de título de la ventana del navegador.

    body: Esta sección contiene todo el contenido visible de la página web, como texto, imágenes, enlaces, formularios, etc. Todo lo que se muestra en el navegador se coloca dentro de este elemento

Etiquetas HTML Más Usadas

  • p: Esta etiqueta se utiliza para representar un párrafo de texto en una página web.
  • h1,h2,h3,h4,h5,h6
  • a
  • img
  • ul,ol,li
  • input
  • form
  • table,td,tr

CSS

CSS es un lenguaje de estilo utilizado para definir y el formato de un documento HTML o XML. Permite controlar el diseño, el color, la tipografía y otros aspectos visuales de una página web.

Mejora de la Legibilidad con CSS

Selección de Fuentes

Elige fuentes legibles y adecuadas para el contenido. Utiliza la propiedad code font-family code para especificar las fuentes en tu página.

Tamaño del Texto

Ajusta el tamaño del texto para que sea cómodo de leer. Utiliza la propiedad code fontsize code para especificar el tamaño del texto.

Espaciado entre Líneas

El espaciado entre líneas adecuado puede mejorar la legibilidad. Utiliza la propiedad code line-height code para controlar el espaciado entre líneas.
